Vespucci (Osmeditor4android)

View on GitHub

This is the first OpenStreetMap-Editor for [http://developer.android.com Android], Codename "Vespucci". If you want to get a first impression about this software, have a look at "Editing with mobile editors" at flossmanuals.net. Also, please see the wiki on github.com.
Feel free to download and share the code.

Note: Recently built APKs are available from google drive: Release , Beta/Test. These are not signed with the same key as the release on googles play store and may require a de-install before installing.
