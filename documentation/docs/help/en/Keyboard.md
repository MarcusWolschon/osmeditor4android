
# Keyboard and mouse support

## Shortcuts

The default shortcuts use `<Ctrl>` as a modifier key, the mappings can be changed in the [Keyboard shortcuts preference](Advanced%20preferences.md#KeyboardÂ§20shortcuts)

Mouse/trackpad/trackball scroll wheels will zoom in and out.

### General ###

  __u__ Undo
  
  __h__ Help
  
  __&lt;Esc&gt;__ Back button, _not remappable_

### Main display ###

  __f__ GPS follow 
  
  __g__ GPS goto 
  
  __d__ Download data for current view (merges with existing) 
  
  __b__ Download notes/bugs for current view (merges with existing) 

  __+__ Zoom in (default with no modifier key)
  
  __Volume +__ Zoom in _not remappable_ can be disabled in the [Advanced preferences](Advanced%20preferences.md)
  
  __-__ Zoom out (default with no modifier key) can be disabled in the [Advanced preferences](Advanced%20preferences.md)
  
  __Volume -__ Zoom out _not remappable_
  
  __Arrow keys__ Pan in the respective direction _not remappable_

### Element selected

  __e__ Edit tags
  
  __i__ Element information
  
  __c__ Copy
  
  __x__ Cut 
  
  __v__ Paste 
  
  __t__ Paste tags 
  
  __r__ Delete
  
  __&lt;Del&gt;__ Delete (without modifier)

### Node selected

  __m__ Merge (if possible)
  
### Way selected
  
  __s__ Square / straighten (if possible)
  
### Multi-select

  __m__ Merge (if possible)
  
  __s__ Square / straighten (if possible)
  
### Path creation

  __u__ Undo
  
  __f__ Follow way (if possible)
  
  __e__ Finish way and start the property editor
  
  __a__ Finish way and if a closed way start the property editor with address prediction, if an open way start the address interpolation modal

### Property editor

  __k__ Keep changes and exit

  __i__ Element information
 
  __c__ Copy
  
  __x__ Cut 
  
  __v__ Paste
