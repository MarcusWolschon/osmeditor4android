Веспуччі є повнофункційним інструментом для редагування даних OpenStreetMap. Це не інструмент для перегляду мап чи прокладання маршрутів. Для повноцінної роботи вам потрібен <a href="http://www.openstreetmap.org/">обліковий запис OpenStreetMap</a>.

Ви можете завантажити дані мапи для певної області та відредагувати їх.  Після редагування ви можете завантажити їх безпосередньо на сервери OSM.

Будь-яку випадкову зміну можна скасувати, перелік всіх змін буде показано для ознайомлення перед завантаженням.  Автозавершення теґів, заготовки, сумісні з JOSM, посилання на перекладені сторінки з описами об”єктів мапи та навіть автозаповнення назв для вулиць поруч допомагають знайти правильні теґи.

Ми рекомендуємо відключити автоматичне оновлення Веспуччі, щоб ви могли завантажувати свої зміни перед оновленням застосунку.

Докладна інформація та документація знаходяться на <a href="http://vespucci.io/">espucci.io</a> та в довідці застосунку.

Будь ласка, не питайте тут про підтримку чи сповіщайте по проблеми, див <a href="http://vespucci.io/tutorials/faq/#why-cant-you-provide-technical-support-on-the-google-play-store-review-section">чому ми не можемо надати підтримку та прийняти проблеми в розділі оглядів PlayStore</a>. Натомість ви можете <a href="http://vespucci.io/help/en/Introduction/#reporting-problems">сповістити про проблему</a> безпосередньо із застосунку, або <a href="https://github.com/MarcusWolschon/osmeditor4android/issues">на GitHub</a>.

OpenStreetMap, OSM та лого зі збільшувальним склом є зареєстрованими торговими маками <a href="https://osmfoundation.org/">OpenStreetMap Foundation</a>. Vespucci є окремим застосунком не пов’язаним OpenStreetMap Foundation.
